# Brave Search MCP Extension - Quick Start

## Installation Steps

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Compile the extension**
   ```bash
   npm run compile
   ```

3. **Package the extension**
   ```bash
   npm run package
   ```

4. **Install in VS Code**
   - Open VS Code
   - Go to Extensions view (Ctrl+Shift+X / Cmd+Shift+X)
   - Click the "..." menu â†’ "Install from VSIX..."
   - Select the generated `.vsix` file

5. **Configure your API key**
   - Open Command Palette (Ctrl+Shift+P / Cmd+Shift+P)
   - Run "Brave Search MCP: Configure API Key"
   - Enter your Brave Search API key

## Development

Press F5 to launch the Extension Development Host and test your changes.

## Usage

Open Copilot Chat, enable agent mode (sparkle icon), and ask questions that require web search!
